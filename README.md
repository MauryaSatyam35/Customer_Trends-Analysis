# Customer Trend Analysis – End-to-End Data Analytics Project

## Overview

This project demonstrates a complete data analytics workflow, starting from raw data processing to business insights visualization. The objective is to analyze customer purchasing behavior and generate actionable insights using Python, SQL, and Power BI.

---

## Dataset

The project uses a **Customer Trend Analysis** dataset containing information related to:

* Customer demographics
* Purchase history
* Product categories
* Review ratings
* Discounts and promotions
* Purchase frequency patterns

The dataset was loaded and processed using Python for further analysis.

---

## Tools & Technologies

* **Python** – Data loading, cleaning, and exploratory data analysis (EDA)
* **Pandas** – Data manipulation and preprocessing
* **PostgreSQL / MySQL / SQL Server** – Data storage and SQL querying
* **SQL** – Business analysis and data extraction
* **Power BI** – Interactive dashboard creation
* **Gamma AI** – Presentation (PPT) generation
* **Pycharm – Development environment

---

## Project Workflow

### 1. Data Loading

* Imported the dataset using Python and Pandas.
* Reviewed dataset structure and summary statistics.

### 2. Exploratory Data Analysis (EDA)

* Analyzed data distributions and customer trends.
* Identified missing values and data quality issues.
* Explored relationships between key variables.

### 3. Data Cleaning & Transformation

* Handled missing values.
* Renamed and standardized column names.
* Created new features such as:

  * Customer age groups
  * Purchase frequency in days
* Removed unnecessary columns and prepared data for analysis.

### 4. SQL Database Integration

* Connected Python with PostgreSQL/MySQL/SQL Server.
* Loaded the cleaned dataset into the database.
* Executed SQL queries to answer business questions and extract insights.

### 5. Power BI Dashboard Development

Created an interactive dashboard to visualize:

* Customer demographics analysis
* Sales and purchase trends
* Product category performance
* Customer review patterns
* Discount and promotional impact
* Purchase frequency insights

### 6. Reporting & Presentation

* Compiled findings into a business report.
* Designed a professional presentation using **Gamma AI** to communicate key insights and recommendations.

## How to Run the Project

### Prerequisites

Install the required Python libraries:

```bash
pip install pandas sqlalchemy psycopg2-binary
```

### Steps

1. Clone this repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

```bash
cd customer-trend-analysis
```

3. Update database credentials in the Python script.

4. Run the data processing script:

```bash
python main.py
```

5. Execute SQL queries in your preferred database system (PostgreSQL/MySQL/SQL Server).

6. Open the Power BI file (`.pbix`) to explore the interactive dashboard.

7. Review the generated report and presentation for business insights.
